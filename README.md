# Website Jenis Batik Indonesia

## Deskripsi
Website yang menampilkan informasi mengenai berbagai jenis batik Indonesia. Website ini dibuat sebagai proyek UTS mata kuliah Web Programming.

## Fitur
- Menampilkan berbagai jenis batik Indonesia
- Informasi detail setiap batik
- Tampilan responsif
- Navigasi sederhana

## Teknologi
- HTML
- CSS
- JavaScript
